# Market Bazaar PWA

Mobile-first offline-first PWA для торговли на базаре: быстрые продажи по весу или по сумме, общие партии остатков, локальная база на Dexie/IndexedDB, установка на телефон.

## Стек

- React + TypeScript + Vite
- Dexie + IndexedDB
- vite-plugin-pwa + Service Worker
- Полностью локальное хранение данных

## Локальный запуск

```bash
npm install
npm run dev
```

Сборка production:

```bash
npm run build
```

## Что уже есть

- Экран быстрой продажи с лентой товаров
- Режимы `ВЕС` и `СУММА`
- Крупная цифровая панель
- Быстрое изменение веса, суммы и цены
- Скидка, округление, расчет сдачи
- Общая партия остатка для нескольких вариантов товара
- Товары, партии, поступления, списания, расходы
- История и базовые отчеты
- Экспорт/импорт JSON
- Демо-данные
- Installable PWA

## Как открыть на телефоне через GitHub и добавить на экран домой

### 1. Загрузить проект на GitHub

```bash
git add .
git commit -m "bazaar pwa mvp"
git push origin main
```

### 2. Включить GitHub Pages

В репозитории:

1. `Settings`
2. `Pages`
3. `Source = GitHub Actions`

В проект уже добавлен workflow `.github/workflows/deploy.yml`, который соберет `dist` и опубликует сайт.

### 3. Открыть ссылку на телефоне

После деплоя GitHub Pages даст ссылку вида:

`https://USERNAME.github.io/REPO/`

Откройте ее на телефоне.

### 4. Добавить на главный экран

#### Android / Chrome

1. Открыть сайт
2. Нажать меню браузера
3. Выбрать `Установить приложение` или `Добавить на главный экран`

#### iPhone / Safari

1. Открыть сайт
2. Нажать `Поделиться`
3. Выбрать `На экран "Домой"`

После первого открытия приложение кэшируется сервис-воркером и дальше может работать офлайн.

## Важно

- GitHub Pages все равно нужен для первой загрузки по ссылке
- После установки приложение работает как локальная PWA
- Данные хранятся только на конкретном устройстве в IndexedDB
- Для переноса данных между телефонами используйте экспорт/импорт JSON
